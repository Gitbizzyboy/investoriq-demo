# InvestorIQ Demo Checklist

## Pre-Demo Setup (5 minutes)
- [ ] Clear browser cache
- [ ] Test all main pages load (/, /map, /pipeline, /social)
- [ ] Add 2-3 properties to pipeline for demo
- [ ] Have 2-3 specific properties ready to click (know their stories)
- [ ] Test one full property analysis flow

## Demo Flow (15 minutes)

### Opening Hook (2 min)
"Every distressed property in the Quad Cities - $3.45M in opportunities mapped and analyzed"

### 1. Property Discovery (4 min)
- Show map with 109 properties
- Click on high-value property (say Milan property ~$25K+ tax debt)
- Walk through property analysis page
- Show ROI calculator in action

### 2. Deal Management (4 min)  
- Add property to pipeline from map
- Show pipeline stages
- Demonstrate note-taking and document upload
- Show LP report generation

### 3. Market Intelligence (3 min)
- Quick tour of market analytics
- Show city-by-city breakdown
- Highlight ROI projections by area

### 4. Professional Network (2 min)
- Show Bryce's profile in social intel
- Click tabs to show expertise areas
- Demo contact/message functionality
